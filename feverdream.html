I got the idea for this HTML and CSS combo from using the Readability add-on. I used it so often that I became (even more) annoyed with the way that so many web pages get written, these days. Web pages are made to be read. Well, they're supposed to be. Instead, too many web pages are so painful to read that some people made hacks to focus on the text, again.

The very existence of an add-on like Readable is an indictment on the web design profession. If your users click on the Readable button while visiting your site, then that button may as well say "The designer sucks and needs to get another job!" I got worried about being in the place of that hypothetical designer, so I decided to cut out the middle man and design my sites to look as if they had already gone through the Readable plugin.

The line lengths are short and the text size is large. This eases reading. It's all pixels, so there's no need to save paper by keeping things small. The sides of the box hug the sides of the essay in order to guide the reader's eyes downward, further easing reading.

I centered everything. In order to keep symmetry, I justified the text so that are no ragged edges on either the left or right sides of the paragraphs. Since lines lengths are so short, the reader should have no problem going to the next line.

Most of the heavy lifting is done in the CSS file. No JavaScript is required to make this work.
